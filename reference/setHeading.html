setHeading
index		  
Name setHeading ()
Description Sets Turtle's heading--the direction the Turtle is facing.
Syntax setHeading (angle);
Parameters
angle  Turtle's new heading, in degrees. (float)
Returns
None  
Related setX()
setY()

getX()
getY()
getHeading()
Example 

    import Turtle.*;
    size(500,500);
    background(255);
    stroke(0);

    Turtle t = new Turtle(this);

    t.right(45);
    t.forward(50);
    t.setHeading(0); 
    t.forward(50);
