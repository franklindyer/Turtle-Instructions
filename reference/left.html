left
index		  
Name left ()
Description Turns the turtle left by an angle.
Syntax left (angle);
Parameters
angle   (float)
Returns None
Related right()
Example 

    import Turtle.*;
    size(500,500);
    background(255);
    stroke(0);

    Turtle t = new Turtle(this);

    //draws a square
    t.forward(100);
    t.left(90);    
    t.forward(100);
    t.left(90);    
    t.forward(100);
    t.left(90);    
    t.forward(100);
    t.left(90);
