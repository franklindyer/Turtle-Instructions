penDown
index		  
Name penDown ()
Description Sets the current pen state to PENDOWN. Turtle will draw as it moves around the screen. The turtle's default pen state is PENDOWN.
Syntax penDown();
Parameters
None  
Returns None
Related penUp()
Example 

    import Turtle.*;
    size(500,500);
    background(255);
    stroke(0);

    Turtle t = new Turtle(this);


    t.forward(50);
    t.penUp();
    t.forward(50);
    t.penDown();
    t.forward(50);
