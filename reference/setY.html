setY
index		  
Name setY ()
Description Sets Turtle's Y coordinate. Creates a line between the previous location and new location (with new Y coordinate).
Syntax setY (y);
Parameters
y  Turtle's new Y coordinate (float)
Returns
None  
Related setX()
setHeading()

getX()
getY()
getHeading()
Example 

    import Turtle.*;
    size(500,500);
    background(255);
    stroke(0);

    Turtle t = new Turtle(this);

    t.right(45);
    t.forward(50);
    t.setY(0);
