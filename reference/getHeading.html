Name getHeading ()
Description Returns Turtle's current heading--the direction the Turtle is currently facing.
Syntax getHeading ();
Parameters
None  
Returns
angle  Turtle's current heading, in degrees. float

Example 

    import Turtle.*;
    size(500,500);
    background(255);
    stroke(0);

    Turtle t = new Turtle(this);
    float x;


    //print Turtle's x coordinate
    t.forward(50);
    x=t.getHeading(); 
    println(x);
